# scala-web-sbt-play-api-param-hello-world

## Description
A call to playframework web api
with a value.

## Tech stack
- scala
- sbt
  - play

## Docker stack
- hseeberger/scala-sbt:11.0.2-oraclelinux7_1.3.5_2.12.10

## To run
`sudo ./install.sh -u`
End points
- http://localhost
- http://localhost/John

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
